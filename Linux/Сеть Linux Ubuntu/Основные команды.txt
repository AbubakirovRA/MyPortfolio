Получение информации о состоянии сети и о сетевых интерфейсах:
просмотр информации о подключениях
ifconfig

посмотреть состояние интерфейсов
nmcli device status

посмотреть интерфейсы докер
docker network ls

типы интерфейсов:
enp - физический сетевой адаптер 
br-... - интерфейс типа мост (bridge) используется docker контейнерами для настройки сети между своими контейнерами
veth - виртуальный интерфейс, также используется docker
lo - локальный интерфейс
